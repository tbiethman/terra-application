# ApplicationPageStatus

The ApplicationPageStatus is used to render a Status View within the application.

When mounted, the ApplicationPageStatus will communicate with an ancestor
[ApplicationPageStatusProvider](/application/terra-application/components/application-page-status-provider) to
present a staus view.

> See the [How to Show Page Status View](/application/terra-application/how-to/show-page-status-view) page for usage information.

## Usage

```jsx
import ApplicationPageStatus from 'terra-application/lib/application-page-status';
```
