# How to Show Page Status View

Status View can be rendered within the framework by rendering an [ApplicationPageStatus](/application/terra-application/components/application-page-status) component. Status Views will be rendered by the closest ancestor [ApplicationPageStatusProvider](/application/terra-application/components/application-page-status-provider). All content rendered within the responding ApplicationPageStatusProvider will be overlaid.

```jsx
import React, { useState } from 'react';
import ApplicationPageStatus from 'terra-application/lib/application-page-status';

const ExampleComponent = () => {
  const [showPageStatus, setShowPageStatus] = useState(false);

  return (
    <div>
      <p>This component toggles a status view that lasts for 3 seconds.</p>
      <p>
        <button
          type="button"
          onClick={() => {
            setShowPageStatus(true);

            setTimeout(() => {
              setShowPageStatus(false);
            }, 3000);
          }}
        >
          Show Status View
        </button>
      </p>
      {showPageStatus && <ApplicationPageStatus message="Demo for showing page status view" variant="no-data" />}
    </div>
  );
};

export default ExampleComponent;
```

Any component that wants to create an status view container for its children can render its own [ApplicationPageStatusProvider](/application/terra-application/components/application-page-status-provider) to scope status view presentation to that container.

> Note: ApplicationPageStatusProviders are provided by [ApplicationBase](/application/terra-application/components/application-base), [ApplicationNavigation](/application/terra-application/components/application-navigation) , [ModalManager](/application/terra-application/components/modal-manager), and the [SlidePanelManager](/application/terra-application/components/slide-panel-manager).
